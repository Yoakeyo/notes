<div>和<span>标签	#<div>和<span>没有语义，他们是一个盒子，用来装内容
	<div></div>		#div是division的缩写，表示分割、分区
		一行只能放一个<div>。
		大盒子
	<span></span>	#span意为跨度、跨距
		一行可以放多个<span>。
		小盒子

图像标签和路径
	图像标签	<img>	#用于定义HTML页面中的图像
		<img src="图像URL" />	#单词image的缩写，意为图像
			#属性：属于该标签的特性
				src		#source的缩写，意为来源
					是<img>标签中的必须属性，用于指定图像文件的路径和文件名		
				alt		
					替换文本，在图像显示不出来的时候用文字替换
				
				title
					提示文本，鼠标放在图像上时，提示文字
				width
					设置图片的宽度
				height
					设置图片的高度
				border
					设置图片的边框粗细
					
超链接标签
	<a href="跳转目标" target"目标窗口的弹出方式">文本或图像</a>	#anchor的缩写，意为锚
		href 	#(必须属性)用于指定链接目标的url地址
		target 	#用于指定链接页面的打开方式，其中_self为默认值，_blank为在新窗口中打开
		
		外部链接
			<a href="http://www.baidu.com">百度</a>
		内部链接
			<a href="相对链接">目标</a>
		空链接
			<a href="#">用#表示空链接</a>	
		下载链接			#如果href里面的地址是一个文件或者压缩包，会下载这个文件
			<a href="压缩包or文件">下载文件</a>
		网页元素的链接	#在网页中的各种网页元素，如图片，文本，表格，音频等都可以添加超链接
			<a href="http://www.baidu.com"><img src="img.jpg" /></a>
		锚点链接		#通过点击链接，快速定位到文本中的某个位置
			<a href="#名字">点击跳转到名字所在的位置</a>
			<h3 id="名字">名字所在的位置</h3>
		阻止链接跳转
			添加 javascript:void(0); 或者 javascript:;
			
		
		
水平线标签(不常用)
	<hr>	#设置一条水平线分隔页面
