就是控制代码按照什么结构顺序来执行

流程控制主要有三种结构，分别是
	顺序结构
		按着代码的先后顺序，依次执行
		是最简单、最基本的流程控制
		没有特定的语法结构
	分支结构
		语法结构:
			if (条件表达式) {
				条件成立执行的代码语句
			}
		三元表达式
			条件表达式 ? 表达式1 : 表达式2
			如果表达式结果为真 则返回表达式1 否则返回表达式2
		switch 语句		//转换、开关
			switch(表达式) {
				case value1:	//选项
					执行语句1;
					break;
				case value2:
					执行语句2;
					break;
				...
				default:
					执行最后的语句;	
			}
		
	循环结构
		for循环:
			for(初始化变量;条件表达式;操作表达式) {
				//循环体
			}
				当条件表达式为真，则执行循环体
		while循环:
			while(条件表达式) {
				//循环体
			}
				当条件表达式为真，则执行循环体
				先判断后执行
		do while循环:
			do {
				//循环体
			} while(条件表达式)
				跟while不同的是，do while先执行循环体，再判断表达式
				先执行后判断
				至少执行一次循环体
		continue
			立即跳出本次循环，继续下一次循环
		break
			立即结束整个循环
