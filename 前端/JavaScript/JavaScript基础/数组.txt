数组:
	就是指一组数据的集合，其中每个数据被称作元素
	在数组中可以存放任意类型的元素
	是一种将一组数据存储在单个变量名下的优雅方式


创建数组
	利用 new 创建数组
		var 数组名 = new Array();
		var arr = new Array(2);		// 创建一个长度为2的数组，里面有2个空的数组元素
		var arr = new Array(2,3);	// 等价于[2,3]
	利用数组字面量创建数组
		var 数组名 = [];


数组的索引
	索引(下标):用于访问数组元素的序号(数组下标从0开始)
		var arr = ['第一个','123',true,'myname']
			索引号:	  0		 1	   2      3    
		数组可以通过索引来访问(获取、得到)、设置、修改对应的数组元素
		可以通过"数组名[索引]"的形式来获取数组中的元素
	返回数组元素的索引号
		从前往后查找
			indexOf(数组元素)
		从后往前查找
			lastIndexOf(数组元素)	
		只返回第一个满足条件的索引号
		如果找不到则返回 -1

数组的长度
	通过 "数组名.length" 可以访问数组元素的数量(数组长度)

检测是否为数组
	1)参数 instanceof Array 可以用来检测是否为数组
	2)Array.isArray(参数);	// H5 新增的方法
	
添加/删除数组元素的方法
	push(参数1...)					末尾添加一个或多个元素
	pop()							删除最后一个元素
	unshift(参数1...)				开头添加一个或多个元素
	shift()							删除第一个元素
	concat()						链接两个或多个数组 不影响原数组
	slice(begin,end)				返回一个被截取的新数组
	splice(第几个开始,要删除的个数)	返回被删除的新数组(会影响原数组)

数组排序
	翻转数组
		reverse()
	数组排序(冒泡排序)
		sort()
		arr.sort(function(a,b)) {
			return a - b;	//升序的顺序排序
			return b - a;	//降序的顺序排序
		}

数组转换为字符串
	1) toString()
		var arr = [1,2,3];
		console.log(arr.toString());	// 1,2,3
	2) join(分割符)
		var arr = [1,2,3]; 
		console.log(arr.join());		// 1,2,3
		console.log(arr.join('-'));		// 1-2-3
		console.log(arr.join('&'));		// 1&2&3
