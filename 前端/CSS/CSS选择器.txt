
CSS基础选择器
	根据不同需求将不同的标签选择出来，就是选择器
	优先级关系：内联样式 > ID 选择器 > 类选择器 = 属性选择器 = 伪类选择器 > 标签选择器 = 伪元素选择器
	标签选择器
		将HTML标签名称作为选择器
		语法:
			标签名｛
				属性1:属性值1;
				属性2:属性值2;
				属性3:属性值3;
			｝
			
	类选择器
		将类作为选择器
		口诀:
			样式点定义	结构类(class)调用
		语法:
			.类名 {
				属性1:属性值1;
				属性2:属性值2;
				属性3:属性值3;
			}
		注意:
			类选择器使用.(英文点号)来标识，后面紧跟类名
			长名称或词组可以使用中横线来为选择器命名
			不使用纯数字、中文等命名，尽量使用英文
			命名要有意义，能一眼看出类名目的
		多类名:
			<div class="类1 类2"></div>
			各个类名间用空格隔开
			
	id 选择器
		语法:
			#id名 {
				属性1:属性值1;
				...
			}
		口诀:
			样式#定义，结构id调用，只能调一次，别人切勿用
	
	通配符选择器
		语法:
			* {
				属性1:属性值1;
				...
			}
		不需要调用，自动给所有元素/元素使用样式
		
CSS复合选择器	#由多个基础选择器通过不同方式组合在一起
	后代选择器(重要)		#又称包含选择器
		语法:
			元素1 元素2 ... {
				样式声明
			}
			表示选择元素1里面的所有元素2(后代元素)		#也可以是 元素1 里的 元素2 里的元素3
			
		元素1 和 元素2 中间用空格隔开
		元素1 是父级，元素2 是子级，最终选择是 元素2
		元素2 可以是儿子，也可以是孙子等，只要是 元素1 的后代即可
		元素1 和 元素2 可以是任意基础选择器
		
	子选择器(重要)	#只能选择作为某元素的最近一级子元素。就是亲儿子元素
		语法:
			元素1 > 元素2 {
				样式声明
			}
			元素1 和 元素2 中间用 > 隔开
		元素1 是父级，元素2 是子级，最终元素的是元素2
	并集选择器(重要)	#选择多组标签，为他们定义相同的样式
		语法:
			元素1,元素2 {
				样式声明
			}
			元素1 和 元素2 之间用,隔开
		通常竖着写
		最后一个选择器不加括号
		任何形式的选择器都可以作为并合选择器的一部分
			例如:
			ul,
			li,
			.test a {
				color:blue;
			}
			选择 ul 和 li 和 .div下的a
			
	伪类选择器
		链接伪类选择器
			语法:
				a:link		#选择所有未被访问的链接
				a:visited	#选择所有已被访问的链接
				a:hover		#选择鼠标指针位于其上的链接
				a:active	#选择活动链接(鼠标按下未弹起的链接)
			用冒号(:)表示
			注意事项:
				按照LVHA的顺序声明: link -:visited -:hover -:active;
			因为<a>链接在浏览器中有默认样式，所以实际开发中需要单独指定样式
		:focus伪类选择器		#用于选取获得焦点的表单元素
			input:focus
			一般情况下用于<input>之类的表单元素
